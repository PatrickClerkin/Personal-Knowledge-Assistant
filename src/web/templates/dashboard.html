LLM-as-judge evaluation of the full RAG pipeline. Runs each question end-to-end and scores the generated answers on three metrics (Es et al., 2023).

⚠ Each run costs ~7 Claude API calls per question (so an 8-question run is ~56 calls, roughly $0.20). The endpoint is synchronous — expect 2–5 minutes of wall-clock time for a full run.
